Ervaringen

Echte werk ervaringen heb ik niet.

Maar hiervoor kan ik het goed maken, Ik heb stage gedaan bij Sodego. Sodego is een bedrijf in Vorselaar, ze zijn gespecialiseerd in software, deze software is zeer uiteenlopend en niet gericht op 1 doelgroep van klanten. Nee, ze maken verschillende soorten software met verschillende soorten talen onder andere Java, PHP en HTML.

Ook help ik soms is mensen online wanneer nodig met hun software en hardware problemen. Op basis van ervaring heb ik genoeg ervaring als een normale job. Maar zelf heb ik nog niets betaald gedaan. Alles was vrijwillig of voor school, maar telkens genoot ik er echter wel van.

In het middelbaar moest ik voor mijn GIP ook een website maken, hierdoor is Cuisishare ontstaan. Cuisishare is mijn website (niet online gepubliceerd), waar ik hard aan heb gewerkt. De site is een recepten site waar je recepten kan delen met andere mensen, de naam is onstaan door Cuisine (koken) en share(delen) te mixen. Deze bestaat uit API, javascript, HTML, C# en nog meer. Ik ben zelf redelijk trots op de website.
